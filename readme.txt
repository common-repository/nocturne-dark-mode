= Effortlessly transform your Elementor websites into stunning dark-themed versions. =

Nocturne Dark Mode seamlessly integrates with Elementor, allowing you to customize the dark mode settings to match your website's branding and design. Whether you want to create a dark mode version of your entire website or just specific sections, Nocturne Dark Mode gives you the flexibility to do so. Say goodbye to blindingly bright websites and give your users the option to switch to a more comfortable and stylish dark mode with Nocturne Dark Mode for Elementor and Elementor Pro.

= WHY CHOOSE HOW NOCTURNE DARK MODE? =
Unlike other plugins, Noturne Dark Mode adds custom controls to every Elementor widget, making the customization process more smooth an easy. Everything is editable via Elementor editor screen.

= NOCTURNE DARK MODE FREE FEATURES =
1. Easily customize any Elementor widget.
2. Works with sections and containers.
3. **Dark Mode Image Elementor widget** - display different image when dark mode is activate.
4. **Dark Mode Switcher Elementor widget** - place dark mode widget anywhere on the page.
5. Set Dark Mode as the default color scheme.
6. Save user choice every time they visit your site.

= NOCTURNE DARK MODE PRO FEATURES =
1. Elementor Pro compatibility
2. 3 Toggle styles

== Installation ==
1. Activate the plugin and edit any page or template with Elementor.
2. Place Dark Mode Switcher widget anywhere on the page. Usually the best place is your site header.
3. Navigate to Appearance -> Customize -> Dark Mode to change the default settings.

== Frequently Asked Questions ==

= Does it work with all WordPress themes? =
Yes, it works with all WordPress themes.
